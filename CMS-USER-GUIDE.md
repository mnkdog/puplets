# Puplets CMS User Guide

**Welcome!** This guide will help you manage the Puplets website without any coding knowledge.

---

## 📱 Accessing the Admin Panel

1. Go to: **https://your-site.vercel.app/admin** (or `/admin` on your domain)
2. Click **"Login with GitHub"**
3. Authorize the CMS to access your GitHub account
4. You're in! 🎉

---

## 🏠 Admin Dashboard Overview

When you log in, you'll see the main dashboard with these sections:

### Collections (Left Sidebar)
- **Blog Posts** - Write and manage blog articles
- **Pages** - Edit or create custom pages
- **About Page** - Edit the About Us page
- **Products - Collars** - Manage collar products
- **Products - Charms** - Manage individual charms
- **Site Settings** - Update site-wide settings

---

## ✍️ Creating a Blog Post

1. Click **"Blog Posts"** in the sidebar
2. Click **"New Blog Post"** button (top right)
3. Fill in the fields:

   - **Title**: Your blog post title
   - **Publish Date**: When to publish (can schedule for future)
   - **Author**: Your name or "Puplets Team"
   - **Featured Image**: Upload a cover image (optional)
   - **Description**: Short summary (appears in previews)
   - **Body**: Write your content using the rich text editor
   - **Categories**: Add categories (News, Tips, etc.)
   - **Tags**: Add relevant tags
   - **Published**: Toggle ON to publish, OFF to save as draft

4. Click **"Publish"** (or **"Save Draft"**)
5. Done! Your post is live (or saved for later)

### Rich Text Editor Tips

- **Bold**: Select text and click the **B** button
- **Italic**: Select text and click the *I* button
- **Headings**: Use ## for main headings, ### for subheadings
- **Images**: Click the image icon to upload
- **Links**: Select text, click link icon, paste URL

---

## 📄 Editing the About Page

1. Click **"About Page"** in the sidebar
2. Click on **"About"**
3. Edit any field:
   - **Title**: Page title
   - **Heading**: Main heading on the page
   - **Image**: Upload a header image
   - **Content**: Edit the page content
4. Click **"Publish"** to save changes
5. Your About page is updated!

---

## 📝 Creating a New Page

1. Click **"Pages"** in the sidebar
2. Click **"New Page"**
3. Fill in:
   - **Title**: Page name
   - **Description**: Brief description for SEO
   - **Featured Image**: Header image (optional)
   - **Body**: Page content
   - **Template**: Choose layout style
4. Click **"Publish"**
5. Your new page is created at `/pages/your-page-name.html`

---

## 🛍️ Managing Products

### Adding a New Collar

1. Click **"Products - Collars"**
2. Click **"New Product"**
3. Fill in:
   - **Product Name**: e.g., "Classic Dog Collar"
   - **Description**: What makes it special
   - **Base Price**: Starting price
   - **Images**: Upload multiple product photos
   - **Sizes**: Add size options (XS, S, M) with measurements and prices
   - **Colors**: Add available colors
   - **In Stock**: Toggle ON/OFF
4. Click **"Publish"**

### Adding/Editing Charms

1. Click **"Products - Charms"**
2. Click **"New Charm"** or edit existing
3. Fill in:
   - **Charm Name**: e.g., "Strawberry"
   - **Description**: Brief description
   - **Price**: Usually £3.99
   - **Image**: Upload charm photo
   - **In Stock**: Toggle ON/OFF
   - **Order**: Number (0 = first, 1 = second, etc.)
4. Click **"Publish"**

---

## ⚙️ Updating Site Settings

1. Click **"Site Settings"**
2. Click **"General Settings"**
3. Edit:
   - **Site Title**: Your site name
   - **Site Description**: Short description for search engines
   - **Footer Text**: Copyright text
   - **Navigation**: Add/remove/reorder menu links
   - **Social Media**: Add your social media URLs
   - **Shipping**: Update shipping info
4. Click **"Publish"**

---

## 🖼️ Uploading Images

### Best Practices

- **Size**: Keep images under 2MB
- **Format**: Use JPG for photos, PNG for graphics
- **Dimensions**: 
  - Blog featured images: 1200x630px
  - Product images: 1000x1000px
  - Charm images: 500x500px

### How to Upload

1. Click the **"Upload"** button in any image field
2. Choose from:
   - **Upload**: Select file from your computer
   - **Media Library**: Choose from previously uploaded images
3. Image uploads automatically!

---

## 📅 Publishing & Drafts

### Save as Draft
- Toggle **"Published"** to **OFF**
- Click **"Save Draft"**
- Post is saved but NOT visible on the website

### Publish Immediately
- Toggle **"Published"** to **ON**
- Click **"Publish"**
- Post goes live instantly

### Schedule for Later
- Set **"Publish Date"** to a future date
- Toggle **"Published"** to **ON**
- Post will appear automatically on that date

---

## 🔄 Editing Existing Content

1. Navigate to the collection (Blog Posts, Pages, etc.)
2. Click on the item you want to edit
3. Make your changes
4. Click **"Publish"** to save
5. Changes appear on the website within 1-2 minutes

---

## 🗑️ Deleting Content

1. Open the item you want to delete
2. Click the **"Delete"** button (usually top right)
3. Confirm deletion
4. Item is removed from the website

⚠️ **Warning**: Deleted content can't be recovered easily!

## 🆘 Troubleshooting

### "Changes aren't showing on the website"
- Wait 1-2 minutes for the site to rebuild
- Clear your browser cache (Ctrl+F5 or Cmd+Shift+R)
- Check if post is set to **Published: ON**

### "I can't log in"
- Make sure you're using the correct GitHub account
- Check you've authorized the CMS app
- Try logging out and back in

### "Image upload failed"
- Check image is under 2MB
- Try a different image format (JPG or PNG)
- Make sure filename doesn't have special characters

### "I made a mistake!"
- Go back to the item and edit it
- Changes are saved in GitHub - you can view history
- Ask for help if you need to restore an old version

## 💡 Tips & Best Practices

### Blog Writing
- **Write catchy titles** - they appear in search results
- **Use images** - posts with images get more views
- **Keep paragraphs short** - easier to read on mobile
- **Add categories/tags** - helps organize content
- **Proofread** - save as draft, review, then publish

### Product Updates
- **Update "In Stock"** when items sell out
- **Add multiple images** - customers want to see details
- **Be descriptive** - explain what makes products special
- **Keep prices updated** - check regularly

### Images
- **Name files clearly**: "strawberry-charm.jpg" not "IMG_1234.jpg"
- **Compress large images** - use tinypng.com first
- **Use good lighting** - clear, well-lit photos work best
